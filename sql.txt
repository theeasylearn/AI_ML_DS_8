1) How to insert row in table 
Ans to insert row in table, we use INSERT SQL Command
    there are 2 method to insert row into table. 
    
    1st method syntax 
    insert into <tablename> values (value1,value2,.....,value-n)

    Example 
    -----------------
    insert into category values (0,'Watch','watch.jpg','smart watch for mans and womans',now(),0)

    2nd method syntax 
    insert into <tablename> (field-1,field-2,....field-N) values (value1,value2,.....,value-n)

    Example 
    -----------------
    insert into category (name,photo,detail) values ('Computer','computer.jpg','desktop & laptops')


2) How to delete row(s) from table?

delete all rows 
delete from <tablename>
example 
delete from category 

delete only specific row (as per condition)
delete from <tablename> where condition
delete category whose id is 1 
delete from category where id = 1

delete category whose where id is above 3 
delete from category where id>3

we can use any relational operators  in condition 
= != < > <= >=
we can use also any logical operators in condition as well 
and or not
